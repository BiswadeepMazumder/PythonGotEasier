Week 2: Data Structures in Python

2.1 Lists

Lists are a fundamental data structure in programming that allow you to store multiple items of data in a single variable. They are used to represent collections of related data, such as a list of numbers, a list of names, or a list of objects.

2.1.1 To create a list, you use square brackets [] and separate each item with a comma. For example:


        my_list = [1, 2, 3, 4, 5]
        names = ["Alice", "Bob", "Charlie", "Dave"]
        mixed_list = [1, "hello", True, 3.14]

2.1.2 You can access individual items in a list by their index, which is the position of the item in the list. The first item in a list has an index of 0, the second item has an index of 1, and so on. To access an item, you use square brackets [] and the index of the item you want. For example:


        my_list = [1, 2, 3, 4, 5]
        print(my_list[0])  # prints 1
        print(my_list[2])  # prints 3

2.1.3 You can also use slicing to access a range of items in a list. Slicing is done by specifying the start and end indices separated by a colon. For example:


        my_list = [1, 2, 3, 4, 5]
        print(my_list[1:3])  # prints [2, 3]
        print(my_list[:3])   # prints [1, 2, 3]
        print(my_list[3:])   # prints [4, 5]

2.1.4 To modify a list, you can assign a new value to an existing item using indexing. For example:


        my_list = [1, 2, 3, 4, 5]
        my_list[2] = 10
        print(my_list)  # prints [1, 2, 10, 4, 5]

2.1.5 You can also add new items to a list using the append() method. This adds a new item to the end of the list. For example:


        my_list = [1, 2, 3, 4, 5]
        my_list.append(6)
        print(my_list)  # prints [1, 2, 3, 4, 5, 6]

2.1.6 You can insert a new item at a specific index using the insert() method. This shifts the existing items to the right to make room for the new item. For example:


        my_list = [1, 2, 3, 4, 5]
        my_list.insert(2, 10)
        print(my_list)  # prints [1, 2, 10, 3, 4, 5]

2.1.7 Finally, you can remove an item from a list using the del keyword or the remove() method. The del keyword removes the item at a specific index, while the remove() method removes the first occurrence of a value: 


        my_list = [1, 2, 3, 4, 5]
        del my_list[2]
        print(my_list)  # prints [1, 2, 4, 5]

        my_list = [1, 2, 3, 4, 5]
        my_list.remove(3)
        print(my_list)  # prints [1, 2, 4, 5
