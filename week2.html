2.1 Lists

2.1.1 What is a List in python?

Lists are a fundamental data structure in programming that allow you to store multiple items of data in a single variable. They are used to represent collections of related data, such as a list of numbers, a list of names, or a list of objects.

2.1.2 Create a list in python

To create a list, you use square brackets [] and separate each item with a comma. For example:


        my_list = [1, 2, 3, 4, 5]
        names = ["Alice", "Bob", "Charlie", "Dave"]
        mixed_list = [1, "hello", True, 3.14]

2.1.2 Accessing elements of a list

You can access individual items in a list by their index, which is the position of the item in the list. The first item in a list has an index of 0, the second item has an index of 1, and so on. To access an item, you use square brackets [] and the index of the item you want. For example:


        my_list = [1, 2, 3, 4, 5]
        print(my_list[0])  # prints 1
        print(my_list[2])  # prints 3

2.1.3 Slicing List

You can also use slicing to access a range of items in a list. Slicing is done by specifying the start and end indices separated by a colon. For example:


        my_list = [1, 2, 3, 4, 5]
        print(my_list[1:3])  # prints [2, 3]
        print(my_list[:3])   # prints [1, 2, 3]
        print(my_list[3:])   # prints [4, 5]

2.1.4 Modify list contents

To modify a list, you can assign a new value to an existing item using indexing. For example:


        my_list = [1, 2, 3, 4, 5]
        my_list[2] = 10
        print(my_list)  # prints [1, 2, 10, 4, 5]

2.1.5 Adding new elements in a list

You can also add new items to a list using the append() method. This adds a new item to the end of the list. For example:


        my_list = [1, 2, 3, 4, 5]
        my_list.append(6)
        print(my_list)  # prints [1, 2, 3, 4, 5, 6]

2.1.6 Insert element at any location in a List

You can insert a new item at a specific index using the insert() method. This shifts the existing items to the right to make room for the new item. For example:


        my_list = [1, 2, 3, 4, 5]
        my_list.insert(2, 10)
        print(my_list)  # prints [1, 2, 10, 3, 4, 5]

2.1.7 Removing elements from a List

You can remove an item from a list using the del keyword or the remove() method. The del keyword removes the item at a specific index, while the remove() method removes the first occurrence of a value: 


        my_list = [1, 2, 3, 4, 5]
        del my_list[2]
        print(my_list)  # prints [1, 2, 4, 5]

        my_list = [1, 2, 3, 4, 5]
        my_list.remove(3)
        print(my_list)  # prints [1, 2, 4, 5

2.1.8 Looping through a List

You can loop through the list items by using for and while loop:


               #Print all items in the list, one by one using for loop:
                
               thislist = ["apple", "banana", "cherry"]
                for x in thislist:
                  print(x)

2.1.10 Why is List used in python

  1. Flexibility: Lists can hold any combination of data types and can be easily modified (added to, removed from, or reordered). This makes lists a highly flexible and customizable data structure that can be used in a wide range of applications.
  2. Efficiency: Lists are highly efficient for storing and accessing ordered data, and many built-in functions and methods make it easy to work with lists in Python.
  3. Versatility: Lists can be used in a wide range of applications, from simple data storage and retrieval to more complex operations such as sorting, filtering, and mapping data.
  4. Compatibility: Python's built-in support for lists and their associated functions and methods make them highly compatible with other Python libraries and frameworks.

2.2 Tuple

2.2.1 What is a Tuple in python?

Tuples are another type of data structure in Python that are similar to lists, but with one key difference: tuples are immutable, whereas lists are mutable. This means that once you create a tuple, you cannot change its values.

2.2.2 Create a Tuple in python

To create a tuple, you use parentheses () and separate each item with a comma. For example:


            my_tuple = (1, 2, 3, 4, 5)
            names = ("Alice", "Bob", "Charlie", "Dave")
            mixed_tuple = (1, "hello", True, 3.14)

2.2.3 Accessing elements in a Tuple

You can access individual items in a tuple by their index, which works the same way as it does for lists. The first item in a tuple has an index of 0, the second item has an index of 1, and so on. To access an item, you use square brackets [] and the index of the item you want. For example:


            my_tuple = (1, 2, 3, 4, 5)
            print(my_tuple[0])  # prints 1
            print(my_tuple[2])  # prints 3

2.2.3.1 Print the second item in the tuple:


            thistuple = ("apple", "banana", "cherry")
            print(thistuple[1]) #Output: apple

2.2.3.2 Negetive Indexing in tuple:

-1 refers to the last item, -2 refers to the second last item etc.


            thistuple = ("apple", "banana", "cherry")
            print(thistuple[-1]) #Output: cherry

2.2.3.2 Range of Index in tuple:

You can specify a range of indexes by specifying where to start and where to end the range.
When specifying a range, the return value will be a new tuple with the specified items.


            thistuple = ("apple", "banana", "cherry", "orange", "kiwi", "melon", "mango")
            print(thistuple[2:5])   #Output: ('cherry', 'orange', 'kiwi')

2.2.4 Adding elements in a tuple

You are allowed to add tuples to tuples, so if you want to add one item, (or many), create a new tuple with the item(s), and add it to the existing tuple


            thistuple = ("apple", "banana", "cherry")
            y = ("orange",)
            thistuple += y

            print(thistuple)

            #Output
            ('apple', 'banana', 'cherry', 'orange')

Note: When creating a tuple with only one item, remember to include a comma after the item, otherwise it will not be identified as a tuple.

2.2.5 Slicing tuple

You can also use slicing to access a range of items in a tuple, just like you can with lists. Slicing is done by specifying the start and end indices separated by a colon. For example:


            my_tuple = (1, 2, 3, 4, 5)
            print(my_tuple[1:3])  # prints (2, 3)
            print(my_tuple[:3])   # prints (1, 2, 3)
            print(my_tuple[3:])   # prints (4, 5)

However, as mentioned earlier, tuples are immutable, so you cannot modify their values directly. If you try to assign a new value to an item in a tuple, you will get a TypeError. For example:


            my_tuple = (1, 2, 3, 4, 5)
            my_tuple[2] = 10  # raises TypeError: 'tuple' object does not support item assignment

If you need to modify a tuple, you will need to create a new tuple with the desired values. For example:


            my_tuple = (1, 2, 3, 4, 5)
            new_tuple = my_tuple[:2] + (10, 20) + my_tuple[3:]
            print(new_tuple)  # prints (1, 2, 10, 20, 5)

2.2.7 Why is tuple used in python

While tuples may seem less versatile than lists, they have their own advantages. For example, tuples can be used as keys in dictionaries, whereas lists cannot. Additionally, because tuples are immutable, they are often more efficient to use than lists in certain situations.

  1. Tuples are immutable, meaning their values cannot be modified once they are created. This makes them useful for representing fixed values, such as coordinates or settings, that should not be changed during program execution.
  2. Tuples can be used as keys in dictionaries, while lists cannot. This is because tuples are hashable, meaning their contents can be converted into a unique integer value that can be used as an identifier.
  3. Tuples can be used to eturn multiple values from a functionr, which can be unpacked and assigned to separate variables. This makes it easy to write concise and readable code that returns complex data.

2.3 Dictionaries

2.3.1 What is a Dictionary in python?

Dictionaries are used to store data values in key:value pairs. A dictionary is a collection which is ordered, changeable and do not allow duplicates.

2.3.2 What make dictionaries different from List and Tuple?

  1. Dictionary Items:Dictionary items are ordered, changeable, and does not allow duplicates.
    Dictionary items are presented in key:value pairs, and can be referred to by using the key name.
  2. Changeable:Dictionaries are changeable, meaning that we can change, add or remove items after the dictionary has been created.
  3. Duplicates Not Allowed:Dictionaries cannot have two items with the same key.
Note:As of Python version 3.7, dictionaries are ordered. In Python 3.6 and earlier, dictionaries are unordered.

2.3.3 Create a Dictionary in python

Adding an item to the dictionary is done by using a new index key and assigning a value to it:

2.3.10 Advantages of Dictionaries

  1. Fast access and retrieval of values: Dictionaries use a hash table to store their key-value pairs, which allows for fast access and retrieval of values based on their keys.
  2. Easy data organization: Dictionaries allow you to organize data using a key-value pair system, which is intuitive and easy to understand. This makes it easier to manage and manipulate data.
  3. Flexible and versatile: Dictionaries can store any type of data, including integers, floats, strings, lists, and even other dictionaries. This flexibility makes them a powerful tool for data processing and manipulation.
  4. Efficient memory usage: Dictionaries use less memory than other data structures, such as lists, making them more efficient for storing and manipulating large amounts of data.
  5. Easy to modify: Dictionaries are mutable, which means you can easily add, remove, or modify items within the dictionary. This makes it easy to update and manage data as needed.

2.4 Sets

2.4.1 What is a set in python?

A Set in Python programming is an unordered collection data type that is iterable, mutable and has no duplicate elements.

2.4.2 Some distinct properties of sets

  1. Sets are written with curly brackets.
  2. Set items can appear in a different order every time you use them, and cannot be referred to by index or key.
  3. Once a set is created, you cannot change its items, but you can remove items and add new items.
  4. Sets cannot have two items with the same value.
  5. The values True and 1 are considered the same value in sets, and are treated as duplicates.

2.4.2 Create a set in python

To create a set in Python, you can use curly braces ({}) or the set() function. For example:

          
            my_set = {1, 2, 3, 4, 5}
            another_set = set(['apple', 'banana', 'orange'])

2.4.2 Add element to a set in python

To add one item to a set use the add() method. For example:

          
            thisset = {"apple", "banana", "cherry"}
            thisset.add("orange")
            print(thisset)

2.4.3 Add 2 Sets in python

To add items from another set into the current set, use the update() method. For example:

          
            thisset = {"apple", "banana", "cherry"}
            thisset.add("orange")
            print(thisset)
**NOTE: The object in the update() method does not have to be a set, it can be any iterable object (tuples, lists, dictionaries etc.).

2.4.4 Access Set items in python

You cannot access items in a set by referring to an index or a key.
But you can loop through the set items using a for loop, or ask if a specified value is present in a set, by using the in keyword.
Loop through the set, and print the values, For example: 

          
            thisset = {"apple", "banana", "cherry"}

            for x in thisset:
              print(x) 
              
            #Check if "banana" is present in the set:

            print("banana" in thisset) #Output: True

2.4.5 Loop through a set in python

You can loop through the set items by using a for> loop: For example:

          
            thisset = {"apple", "banana", "cherry"}

            for x in thisset:
              print(x)

2.4.6 Remove item from a set in python

To remove an item in a set, use the remove(), or the discard() method. For example:

          
              thisset = {"apple", "banana", "cherry"}

              thisset.remove("banana")
              print(thisset)  
              
              thisset.discard("banana")
              print(thisset)            # Note: If the item to remove does not exist, discard() will NOT raise an error.
