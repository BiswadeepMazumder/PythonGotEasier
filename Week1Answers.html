Question 10:

Write a program that takes a user's input and checks if it is a valid time (in format "hh:mm:ss").

Answer:


    import re

    time = input("Enter a time (in format 'hh:mm:ss'): ")
    
    pattern = re.compile("^([01]?[0-9]|2[0-3]):[0-5][0-9]:[0-5][0-9]$")
    
    if pattern.match(time):
        print("Valid time!")
    else:
        print("Invalid time.")

Question 3:

Write a program that takes a user's input and calculates the factorial of that number using a while loop.

Answer:


    n = int(input("Enter a number: "))
    fact = 1
    while n > 0:
        fact *= n
        n -= 1
    print("Factorial =", fact)

Answer:

    number = random.randint(1, 10)

            print("Too low!")

Question 7:

Write a program that takes a user's input and prints out the reverse of the input using a while loop.

    n = input("Enter a number: ")

    i = len(n) - 1
    while i >= 0:

        i -= 1
    print("Reverse =", rev)

Question 10:

Write a program that takes a user's input and checks if it is a palindrome using a while loop.

Answer:


    n = input("Enter a word: ")
    i = 0
    j = len(n) - 1
    is_palindrome = True
    while i < j:
        if n[i] != n[j]:
            is_palindrome = False
            break
        i += 1
        j -= 1
    if is_palindrome:
        print(n, "is a palindrome")
    else:
        print(n, "is not a palindrome")
