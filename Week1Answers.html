Question 6:

Write a program that takes a user's input and checks if it is a perfect square, perfect cube, or neither.

Answer:

Here is the Python code to solve the problem:


            number = int(input("Enter a number: "))
            square_root = number ** 0.5
            cube_root = number ** (1/3)
            if square_root == int(square_root):
                print(number, "is a perfect square.")
            elif cube_root == int(cube_root):
                print(number, "is a perfect cube.")
            else:
                print(number, "is neither a perfect square nor a perfect cube.")

Question 10:

Write a program that takes a user's input and checks if it is a valid time (in format "hh:mm:ss").

Answer:


    import re

    time = input("Enter a time (in format 'hh:mm:ss'): ")
    
    pattern = re.compile("^([01]?[0-9]|2[0-3]):[0-5][0-9]:[0-5][0-9]$")
    
    if pattern.match(time):
        print("Valid time!")
    else:
        print("Invalid time.")
